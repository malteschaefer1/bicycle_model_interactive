# Code of Conduct

## Our Standard

This project is committed to a respectful, inclusive, and constructive community.

Expected behavior:

- Be respectful and professional.
- Focus on ideas and evidence, not personal attacks.
- Assume good intent and ask clarifying questions.
- Accept feedback and correct mistakes.

Unacceptable behavior:

- Harassment, hate speech, or discrimination.
- Personal insults, intimidation, or doxxing.
- Deliberate disruption of discussions.
- Publishing private information without consent.

## Scope

This applies to all project spaces, including issues, pull requests, discussions, and reviews.

## Reporting

If you experience or witness unacceptable behavior, contact repository maintainers via GitHub Issues or GitHub Discussions.
If needed, use a private security advisory channel for sensitive reports.

## Enforcement

Maintainers may remove comments, reject contributions, or block participants for violations.
