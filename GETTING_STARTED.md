# Getting Started / Schnellstart

## English

### Requirements

- Node.js 22+ (tested with Node 23)
- A modern browser
- A shell (macOS/Linux/Windows)

### Start The App

```bash
open model/bicycle_model.html
```

If `open` is unavailable, open the file manually in your browser.

### Run Validation And Tests

```bash
./scripts/run_checks.sh
```

### Typical First 5 Minutes

1. Load the app.
2. Move `v` and `delta` sliders.
3. Watch status (`ok`, `warning`, `invalid`).
4. Check the reference-case self-test table.
5. Run `./scripts/run_checks.sh`.
